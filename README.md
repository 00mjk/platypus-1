# qiskit-textbook-edu

Qiskit Textbook edu exploration

## pre-requisites

1. Python environment with `textbook_converter` requirements installed

    ```
    $ cd qiskit-textbook-edu/textbook-converter
    $ pip install -r requirements.txt
    ```

## setup

```
$ npm install
$ npm run build
$ npm start
```

http://localhost:5000/
